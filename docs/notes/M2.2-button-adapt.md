# M2.2 按钮样式迁移与适配

## 改动内容

### 1. 右侧阅读器视图切换按钮 → 底部

**来源**: `cyn/feat-renderer` 分支的 "show raw markdown / show rendered" 按钮格式

**改动文件**: `src/renderer/features/feeds/EntryDetail.tsx`

- 将 Cleaned / Markdown / Source HTML 三个视图切换按钮从内容区**顶部**移至**底部**
- 样式类从 `btn-view-mode` 改为 `btn-toggle-raw`

**改动文件**: `src/renderer/index.css`

- 新增 `.entry-detail-view-controls`：flex 居中容器，底部带分割线
- `.btn-toggle-raw`：从 `display: block` 改为内联按钮样式
- 新增 `.btn-toggle-raw.active`：选中态 accent 背景 + 白色文字
- 新增 `.btn-toggle-raw:disabled`：Source HTML 不可用时的样式

### 2. 左侧 Feed 编辑/删除按钮 → hover 显现

**改动文件**: `src/renderer/features/feeds/FeedList.tsx`

- 将 ✎(编辑) / ✕(删除) 按钮放入 `<button className="feed-item">` 内部
- 位于 feed 名称之后、unread count 之前
- 添加 `e.stopPropagation()` 防止点击按钮触发 feed 选中

**改动文件**: `src/renderer/index.css`

- `.feed-item-action`：20×20px 小型图标按钮，与 feed row 内容区高度匹配
- `.feed-item-actions`：默认 `display: none`，hover 时 `display: flex`
- 删除按钮 hover 时变红色

## 关键设计决策

| 决策 | 选项 | 选择 |
|---|---|---|
| 显现方式 | opacity / display | `display: none`（不占位）|
| 按钮尺寸 | 32px / 24px / 20px | **20px**（匹配 36px row - 16px padding）|
| 按钮位置 | feed-item 外部/内部 | **内部**（利用 flex 布局自动排列）|
